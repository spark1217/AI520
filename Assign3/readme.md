#### How to run the code
Unzip the folder
Run 'NQueens.py' by inputting command.
The output will be saved in 'Solutions/' folder, and the folder will be created if not already exists. The output file has the title in '<FOR/MAC>_<n>.txt' format.  
It will read the chosen algorithm and the size of the board.

To run the program, run command:
python3 NQueens.py <for/mac> <n>
For example,
python3 NQueens.py for 4
python3 NQueens.py mac 5
